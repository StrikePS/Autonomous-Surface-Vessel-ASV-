GNSS - Global Navigation Satellite System 

Gives me the lat, lon, alt measurement (lla frame)

For localization and local measurement, we require NED (north east down) frame
Frame conversion is simple via some math. 
Frame transformation is done by elloptical earth shape. semi major axis to the equator, semi minor axis to the pole.

EarthA = semi major axis
EarthB = semi minor axis
EarthE = 1 - (EarthB/EarthA)^2 (eccentricity)

Transformation Process: 
lla to ECEF (Earth centred, Earth fixed coordinate frame) : Output - x, y, z
ECEF to NED : Output - 3by1 ned

x = (N+h)*cos(phi)*cos(theta)
y = (N+h)*cos(phi)*sin(theta)
z = ((N-IQ)+h)*sin(phi)
N = (EarthA^2)/sqrt((EarthA*cos(phi))^2 + (EarthB*sin(phi))^2)
IQ = EarthE^2 * N
phu = lat
theta = lon
h = alt

To ned using reference ecef frame.
